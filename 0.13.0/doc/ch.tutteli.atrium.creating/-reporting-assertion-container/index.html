ReportingAssertionContainer

interface ReportingAssertionContainer<T> : RootExpect<T> (source)
Deprecated: Use RootExpect instead; will be removed with 1.0.0

Represents a container for Assertions and offers the possibility to addAssertions which are reported in case they do not hold.

Parameters

T - The type of the subject of this Assert.

Types

AssertionCheckerDecorator

 interface AssertionCheckerDecorator<T>

Represents a decorator for AssertionChecker which passes a specified assertion verb, representation and assertions to AssertionChecker.check for the given maybeSubject.
