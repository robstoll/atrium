newAtMostChecker

abstract fun newAtMostChecker(times: Int, nameContainsNotFun: String, atMostCall: (Int) -> String): IterableContains.Checker (source)

Creates an IterableContains.Checker which verifies that an expected entry is contained at most times in the Iterable.

Parameters

times - The number which the check uses to compare against the actual number of times an expected entry is found in the Iterable.

nameContainsNotFun - The function which should be used instead of atMostCall when times equals to zero.

atMostCall - The function which was used and should not be used if times equals to zero.

Exceptions

IllegalArgumentException - In case times is smaller than 1.
