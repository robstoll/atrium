merge

fun merge(options: ExpectOptions<T>): ExpectOptions<T> (source)

Merges the given options with this object creating a new ExpectOptions where defined properties in options will have precedence over properties defined in this instance.

For instance, this object has defined representationInsteadOfSubject (meaning it is not null) and the given options as well, then the resulting ExpectOptions will have the representationInsteadOfSubject of options.
