isSameAs

infix fun <T : Any> Assert<T>.isSameAs(expected: T): AssertionPlant<T> (source)
Deprecated: Switch from Assert to Expect; will be removed with 1.0.0

Makes the assertion that the Assert.subject is the same instance as expected.

This method might enforce in the future, that expected has to be the same type as Assert.subject. Currently the following is possible: assert(1).isSameAs(1.0)

Exceptions

AssertionError - Might throw an AssertionError if the assertion made is not correct.

Return
This plant to support a fluent API.
