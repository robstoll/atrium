RootAssertionGroupType

object RootAssertionGroupType : AssertionGroupType (source)

The AssertionGroupType for AssertionGroups which are the root of all assertions.

Or in other words, it is the AssertionGroupType for those AssertionGroups which are created when a ReportingAssertionPlant is checking the added Assertions. Its AssertionGroup.description corresponds to the assertion verb used (e.g, assert).
