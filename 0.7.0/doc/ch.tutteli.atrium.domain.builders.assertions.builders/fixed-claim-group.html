fixedClaimGroup

val AssertionBuilder.fixedClaimGroup: FixedClaimAssertionGroupTypeOption (source)

Builder to create an AssertionGroup whose AssertionGroup.holds is fixed (not determined based on its AssertionGroup.assertions).

The intended use case is if all AssertionGroup.assertions are AssertionGroups with an ExplanatoryAssertionGroupType. Such groups always return true for AssertionGroup.holds but you might want to explain a complex failing assertion with those groups. In such a use case this builder is your choice.
