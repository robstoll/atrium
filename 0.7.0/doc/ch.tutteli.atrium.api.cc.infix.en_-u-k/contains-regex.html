containsRegex

infix fun <T : CharSequence> Assert<T>.containsRegex(pattern: String): AssertionPlant<T> (source)
Deprecated: Use pendant from package en_GB; will be removed with 1.0.0

Makes the assertion that AssertionPlant.subject contains a sequence which matches the given pattern.

It is a shortcut for to contain atLeast 1 regex pattern.

Exceptions

AssertionError - Might throw an AssertionError if the assertion made is not correct.

Return
The AssertionPlant for which the assertion was built to support a fluent API.
